Karsten allows you to open collection files in a collection window or as a slide show directly from the command line. The following options are supported:

filepath
Opens the specified collection file in the collection window
-normal filepath
Starts a slide show of the specified file in a window
-maximize filepath
Starts a slide show of the specified file in full screen mode
-noframe filepath
Starts a slide show of the specified file in a borderless window
-wpcont filepath
Starts a slide show of the specified file on the desktop wallpaper
-wponce filepath
Selects a random item out of the specified file and displays it on the wallpaper

Enclose the file path in double quotes if it contains spaces.
